# SolMate Sound Effects

## Required Sound Files
Place the following sound files in this directory:

| File | Description | Duration | Notes |
|------|-------------|----------|-------|
| click.mp3 | Button click | ~50ms | Soft, subtle |
| move.mp3 | Chess piece move | ~100ms | Wood tap sound |
| capture.mp3 | Piece capture | ~150ms | Slightly stronger impact |
| check.mp3 | King in check | ~200ms | Alert-like tone |
| castle.mp3 | Castling move | ~150ms | Double tap sound |
| promote.mp3 | Pawn promotion | ~300ms | Ascending tone |
| checkmate.mp3 | Checkmate | ~500ms | Triumphant/dramatic |
| win.mp3 | Game won | ~800ms | Victory fanfare |
| lose.mp3 | Game lost | ~500ms | Somber tone |
| draw.mp3 | Game draw | ~400ms | Neutral conclusion |
| timeout.mp3 | Time expired | ~300ms | Clock/alarm sound |
| error.mp3 | Invalid action | ~100ms | Error buzz |
| chat.mp3 | QuickChat received | ~150ms | Notification ping |

## Sound Guidelines
- Format: MP3 or OGG (MP3 preferred for compatibility)
- Sample rate: 44100 Hz
- Bitrate: 128 kbps minimum
- Loudness: Normalized, not too loud
- Keep files small (< 50KB each) for fast loading
